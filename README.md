# Hakenman

Hakenmanは派遣社員の勤務表作成サポートアプリです。

## Features

* 簡単に勤務表作成
* 作成した勤務表をCSVでエクスポート
* AppleWatch対応

## Getting Started

1. `git clone https://github.com/dolfalf/Hakenman.git`
2. `cd Hakenman`
3. `pod install` (requires [CocoaPods](https://cocoapods.org))
4. Open `Hakenman.xcworkspace` in Xcode 7.0 or higher.
